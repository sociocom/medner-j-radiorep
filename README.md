# MedNER-J Radiorep

ユーザが入力した医療文書に対し，PRISMアノテーションを付与した結果を表示する．

Clinical NER result viewer based on MedNER-J's radiorep model.

## Requirements

- Python 3.8 (could work with 3.6+ but not tested)
- mojimoji
- [MedNER-J](https://github.com/sociocom/MedNER-J.git)
- Flask
